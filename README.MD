### klikBCA scraper

- Selenium
- Python 3.7

#### How?

- install python , selenium webDriver, chrome driver
- rename config.py.sample -> config.py , edit necessary info
- pipenv install
- pipenv shell
- python app.py

#### Features

- [x] Login
- [x] Logout
- [x] Balance Inquiry (informasi saldo)
- [x] Account Statement (mutasi rekening)
- [ ] History
